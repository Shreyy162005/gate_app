# 📊 GATE Exam Portal (Django Project)

A web-based **GATE Exam Preparation and Assessment System** built using Django.  
It allows students to attempt tests and enables faculty to track performance using a dashboard with analytics.

---

## 🚀 Features

### 👨‍🎓 Student Side
- User registration & login system
- Attempt subject-wise tests
- Instant result calculation
- View test scores

### 👨‍🏫 Faculty Dashboard
- Total student statistics
- All test attempts tracking
- Student-wise performance monitoring
- Top performers list
- Performance analytics using charts

---

## 📊 Key Highlights

- Built using **Django (Python framework)**
- Secure authentication system
- Real-time score calculation
- Chart-based performance visualization
- Clean and responsive UI using Bootstrap

---

## 🛠️ Tech Stack

- Backend: Django (Python)
- Frontend: HTML, CSS, Bootstrap
- Database: SQLite
- Charts: Chart.js
